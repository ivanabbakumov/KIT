# KIT
Этапы создания новогоднего KIT
## Подготовка макета
Находим в интернете понравившийся нам пример подобного KIT. Скачиваем изображение и загружаем его в графический редактор.
В данном случае используем графредактор Figma, в помощью инструмента перо обводим линии со скаченной картинки.

## Экспорт макета
После создания макета эскпортируем его в формате svg (векторный) для дальнейшей работы в программе CoralDraw

<img width="572" height="538" alt="2025-11-13_22-33-01" src="https://github.com/user-attachments/assets/39545da6-efc7-442a-b960-c32584813e09" />

## Работа в CoralDraw
Открываем следующую программу для работы с нашим макетом и импортируем через меню

<img width="536" height="444" alt="Снимок экрана 2025-11-13 223626" src="https://github.com/user-attachments/assets/ff92f7a6-b5d3-441d-9177-ffc081faec5f" />

После импорта макета подготавливаем его к лазерной резке (перекрашиваем линии резки в красный цвет и выставляем толщину "сверхтонкий абрис", элементы гравировки перекрашиваем в черный)

<img width="1137" height="738" alt="Снимок экрана 2025-11-13 224120" src="https://github.com/user-attachments/assets/642f7bff-f5f5-4b74-bc47-bde47028f7fd" />

Замечаем, что при импорте возникла проблема: не все линии воспринимаются правильно. Редактируем испорченные контуры

<img width="893" height="594" alt="Снимок экрана 2025-11-13 230201" src="https://github.com/user-attachments/assets/7cc3b9ff-9115-483f-a638-72cf469b9e73" />
